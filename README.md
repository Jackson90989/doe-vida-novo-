🩸 Tela de Doação de Sangue - Front-End Básico
Este é um projeto de front-end simples e responsivo, criado utilizando apenas HTML e CSS, com o objetivo de simular uma interface de doação de sangue.

💻 Sobre o Projeto
A tela apresenta:

Um layout limpo e organizado

Estilização feita 100% com CSS e HTML, sem frameworks ou bibliotecas externas

🎯 Objetivo
Este projeto foi desenvolvido com fins educacionais, para praticar:

Estruturação de páginas com HTML

Estilização com CSS puro

Boas práticas de organização de código front-end

📁 Como Visualizar
Clone ou baixe este repositório

Abra o arquivo index.html em qualquer navegador moderno

📌 Melhorias Futuras
Adição de interatividade com JavaScript

Formulário funcional de cadastro de doadores

Responsividade aprimorada

Integração com back-end (futuramente)
